Frequently Asked Questions

How do I add a new menu item?

To add a new menu item:

  1. Go to the Menu page
  2. Click the "Add New Item" button
  3. Fill in all required fields (name, price, category, etc.)
  4. Click "Save Item"
How do I create a new order?

To create a new order:

  1. Navigate to the Orders page
  2. Click the "New Order" button
  3. Select a customer from the dropdown
  4. Choose order type (Delivery, Pickup, Dine-in, or Catering)
  5. Add menu items and quantities
  6. Review the order summary and click "Create Order"
How do I make an unavailable menu item available again?

To reactivate an unavailable menu item:

  1. Go to the Menu page
  2. Click the "Unavailable Only" filter button

How do I update order status?

To update an order's status:

  1. Go to the Orders page
  2. Find the order you want to update
  3. Click the "Update Status" button
  4. Select the new status from the dropdown
  5. Click "Update"
What payment methods are accepted?

We currently support the following payment methods:

  • Cash on delivery
  • Credit/Debit cards
  • Digital wallets
  • Bank transfers

Troubleshooting Guide

Common Issues
  • Page not loading: Refresh your browser or clear cache
  • Can't save menu items: Check all required fields are filled
  • Order creation fails: Ensure customer is selected and items added
  • Buttons not working: Try refreshing the page
Quick Fixes
  • Clear browser cache and refresh
  • Check internet connection
  • Try a different browser
  • Restart the application if needed
